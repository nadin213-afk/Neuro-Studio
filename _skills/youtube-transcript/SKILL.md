# YouTube Transcript — для Reels @psy_eft

Извлечение и обработка транскриптов YouTube для создания контента.
Основной юзкейс: видео англоязычных психологов → фрагмент → перевод → Reels.

## Зависимости

```bash
pip install yt-dlp
```

## Быстрый старт

```bash
# Скачать транскрипт
yt-dlp --write-auto-sub --sub-lang en --skip-download \
  --output "%(title)s.%(ext)s" "URL_ВИДЕО"

# Или через Python
python3 -c "
import subprocess, sys
url = sys.argv[1]
subprocess.run(['yt-dlp', '--write-auto-sub', '--sub-lang', 'en,ru',
  '--skip-download', '-o', '%(title)s.%(ext)s', url])
" "URL_ВИДЕО"
```

## Рабочий процесс для Reels

### Шаг 1: Скачай транскрипт
```bash
yt-dlp --write-auto-sub --sub-lang en --skip-download \
  --print-to-file "%(title)s|%(channel)s|%(duration)s" info.txt \
  -o "transcript" "URL"
```

### Шаг 2: Найди ключевой фрагмент (30–60 сек)
Прочитай транскрипт и найди фрагмент, который:
- Содержит одну чёткую мысль о привязанности / травме / отношениях
- Длится 30–60 секунд (примерно 80–150 слов)
- Имеет эмоциональный "крючок" в начале
- Можно перевести на русский без потери смысла

### Шаг 3: Переведи и адаптируй
Переведи фрагмент на русский, соблюдая voice.md:
- Короткие предложения
- Метафоры природы/тела
- Тон: тёплый, не поучающий
- Добавь валидацию если её нет в оригинале

### Шаг 4: Оформи сценарий Reels

```
---
source_video: [URL]
source_channel: [Автор]
fragment_time: [HH:MM:SS — HH:MM:SS]
---

[0–3 сек: крючок — боль или провокация]
Текст: "..."

[4–15 сек: узнавание]
Текст: "..."

[16–30 сек: объяснение через метафору/психологию]
Текст: "..."

[31–45 сек: выход / надежда]
Текст: "..."

[45–60 сек: CTA]
Текст: "Если узнала себя — напиши мне."
```

### Шаг 5: Сохрани
- Сценарий: reels/reel_NN.txt
- Транскрипт: ideas/transcript_[автор]_[тема].md

## Копирайт-правила

Для легального использования:
- Используй только короткие фрагменты (до 60 сек из длинного видео)
- Всегда указывай автора и источник
- Добавляй свой перевод и комментарий (трансформативное использование)
- Субтитры на русском = новое произведение
- В описании Reels: "По мотивам [автор], перевод и комментарий @psy_eft"
