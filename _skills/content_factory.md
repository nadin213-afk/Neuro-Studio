# Навык: Content Factory (Фабрика контента)

## Назначение
Превращение сырых материалов из _incoming/ в готовый контент для posts/, reels/, telegram/.

## Когда использовать
Когда Надежда говорит:
- "Обработай файл из _incoming"
- "Сделай пост из этого транскрипта"
- "Вот заметка из NotebookLM, преврати в контент"

## Алгоритм

### Шаг 1: Чтение контекста
1. Прочитай _style/voice.md
2. Прочитай _knowledge/expert_base.md
3. Прочитай _knowledge/about_me/about_me.txt

### Шаг 2: Анализ сырья
- Тип материала: транскрипт / заметка / цитата / идея
- Ключевая тема (одна)
- Эмоциональное ядро — какая боль аудитории?
- Теоретическая основа — какой концепт EFT подходит?

### Шаг 3: Создание контента

Для поста:
[Крючок — боль или вопрос, 1-2 строки]
[Валидация — 2-3 строки]
[Объяснение через психологию — 3-5 строк]
[Надежда / действие — 2-3 строки]
Если откликается — пишите в ЛС. Первая встреча со скидкой 50%.
[Вопрос к аудитории]

Для Reels:
[0-3 сек: крючок]
[4-15 сек: узнавание]
[16-30 сек: объяснение через метафору или EFT]
[31-45 сек: надежда]
[45-60 сек: CTA]

Для Telegram (до 600 символов):
[Боль в 1 предложении]
[Валидация + объяснение — 2-3 предложения]
[Мягкий CTA с ссылкой на @psychologicalZN]

### Шаг 4: Проверка
- Нет слов: "я здесь", "я рядом", "эффективный", "уникальный", "успешный"
- Нет лишних восклицательных знаков
- Короткие предложения, абзацы 1-3 строки
- Есть метафора (природа, тело, свет)
- Дуга: боль - валидация - объяснение - надежда
- CTA на месте

---

## Правило разделения EFCT / EFIT

При создании контента ВСЕГДА определи направление:

### EFIT (личная терапия) — posts/efit/, reels/efit/, telegram/efit/
Фокус: внутренняя безопасность, самосострадание, отношение к своим эмоциям,
внутренний критик, защитные стратегии, работа с собой после травмы/потери/переезда.

### EFCT (парная терапия) — posts/efct/, reels/efct/, telegram/efct/
Фокус: циклы в паре, Преследователь и Отстраняющийся,
конфликты, восстановление после измены/травмы в паре.
