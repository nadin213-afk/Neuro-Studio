# Social Media Content — для @psy_eft

Создание контента для Instagram и Telegram психолога Надежды Жеманцевой.
Перед началом ОБЯЗАТЕЛЬНО прочитай _style/voice.md.

## Платформы и их особенности

### Instagram (@psy_eft)

**Формат поста:**
- До 2200 символов (оптимально 800–1500)
- Короткие абзацы, много воздуха
- Карусели: 1 мысль = 1 слайд
- Reels: сценарий 45–60 сек

**Структура поста:**
```
[Крючок — боль или вопрос, 1–2 строки]

[Валидация — 2–3 строки]

[Объяснение — психология, 3–5 строк]

[Надежда / действие, 2–3 строки]

Если откликается — пишите в ЛС. Первая встреча со скидкой 50%.

[Вопрос к аудитории — для комментариев]
```

**Структура Reels:**
```
[0–3 сек: крючок — боль или провокация]
[4–15 сек: "ты узнаёшь себя?"]
[16–30 сек: объяснение через метафору или психологию]
[31–45 сек: выход / надежда]
[45–60 сек: CTA — вопрос, сохрани, напиши мне]
```

**Хештеги:**
Не больше 15. Микс:
- Тематические: #тревожнаяпривязанность #отношениясобой #психологонлайн
- Авторские: #psyeft #надеждажеманцева
- Общие: #психология #терапия

### Telegram (@psychologicalZN)

**Формат:**
- Короче, чем Instagram
- Без хештегов
- Более прямой тон (Telegram = ближе к разговору)
- Можно ссылки в тексте

**Адаптация Instagram → Telegram:**
1. Убери хештеги
2. Сократи на 30–40%
3. Сделай тон чуть более личным
4. Замени "пишите в ЛС" на "напишите мне @psychologicalZN"
5. Можно добавить "от себя" — 1–2 личные строки

**Адаптация Telegram → Instagram:**
1. Разверни мысль подробнее
2. Добавь валидацию (в Telegram она может быть сжата)
3. Добавь вопрос для комментариев
4. Добавь хештеги

## Типы контента

### 1. Экспертный пост
Объясняет психологический механизм.
Цель: показать экспертизу через понимание.

### 2. Пост-узнавание
"Это когда ты..." — читательница видит себя.
Цель: отклик, сохранения, комментарии.

### 3. Метафора
Вся мысль — через один образ (дерево, река, дом).
Цель: запоминаемость, репосты.

### 4. Пост от первого лица
Надежда делится своим опытом или наблюдением.
Цель: близость, доверие.

### 5. Промо-пост
Мягко рассказывает о продукте.
Цель: записи. (Используй навык copywriting)

### 6. Серия постов
3–5 связанных постов на одну тему.
Первый пост — крючок, последний — CTA.

## Контент-план

Формат недельного плана:
```
# Неделя: [дата]

## Понедельник — Instagram
Тип: экспертный
Тема: [тема]
Крючок: [1 строка]
Продукт: нет / EFT / ТТЗ

## Среда — Telegram
Тип: метафора
Тема: [тема]
Адаптация из: нового / поста от [дата]

## Пятница — Instagram + Telegram
Тип: узнавание
Тема: [тема]
```

Сохранять: plans/week_YYYY-MM-DD.md

## Процесс создания

1. Определи тип контента и платформу
2. Прочитай voice.md
3. Напиши черновик по структуре платформы
4. Пропусти через humanizer
5. Предложи 2–3 варианта крючка
6. После выбора — финализируй
7. Если нужна адаптация на вторую платформу — адаптируй
